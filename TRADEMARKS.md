# MathFlow Trademark Policy

"MathFlow" and the MathFlow logo are trademarks of the MathFlow project maintainers.

## Permitted Uses

- Referring to MathFlow in blog posts, articles, and academic papers
- Stating compatibility (e.g., "compatible with MathFlow")
- Non-commercial community projects clearly identified as unofficial

## Prohibited Uses

- Using "MathFlow" in the name of a competing product or service
- Using "MathFlow" in a domain name for a hosted service (e.g., mathflow-cloud.com, hosted-mathflow.io)
- Using the MathFlow logo in a way that suggests official endorsement
- Any use that implies the trademark holder's sponsorship or affiliation

## Forks

You may fork MathFlow under the AGPL v3 license, but forks **must be renamed** and must not use the MathFlow name or logo.
