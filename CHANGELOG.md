## [0.1.0] - 2026-03-01

### Added

- WYSIWYG math editor with live KaTeX rendering (inline and display math)
- 200+ LaTeX snippets with auto-expand, tab-expand, and regex triggers
- Theorem environments (theorem, lemma, proof, definition, corollary, proposition, remark, example) with auto-numbering
- TikZ/tikz-cd diagram rendering via tikzjax
- AI assistance (BYOK) with OpenAI, Anthropic, and Google Gemini support
- SymPy symbolic computation integration
- LaTeX + PDF export with custom preamble support
- 4 themes: Light, Dark, Sepia, Nord
- Offline support via Service Worker + IndexedDB
- Self-hosted backend with PostgreSQL, JWT auth, and full CRUD API
- Docker Compose deployment
